🎂 Таймер дня рождения
Красивый таймер обратного отсчета до вашего следующего дня рождения — бесплатно для всех.

https://ilya1python1.github.io/Birthday-Timer/

✨ Особенности:
-⏳ Обратный отсчет в реальном времени — дни, часы, минуты, секунды
-🎨 4 темы — Темная, Светлая, Минималистичная, Терминальная
-🌍 3 языка — Русский, Украинский, Английский
-🌤 Виджет погоды — текущая погода + прогноз на завтра по GPS
-🎯 Индикатор прогресса — как далеко вы продвинулись в году
-🌟 Загадайте желание — с анимацией конфетти
-📊 Интересные факты — сколько чашек кофе, песен, снов осталось до вашего дня рождения
-💾 Сохраняет ваши настройки — запоминает ваше имя, дату и тему

🚀 Использование:
Просто откройте сайт, нажмите ⚙️ и введите свое имя и дату рождения. Вот и все.
Без регистрации. Без рекламы. Совершенно бесплатно.

🛠 Создано с использованием:
чистого HTML / CSS / JavaScript
API Open-Meteo — данные о погоде
Nominatim — преобразование GPS в название города
Шрифты Google Fonts — Inter, Manrope, JetBrains Mono
